Политика конфиденциальности приложения «Правда или Действия: Игра для парочек 18+»

Эта страница описывает, какие данные собираются при использовании мобильного приложения «Правда или Действия: Игра для парочек 18+» (далее — «Приложение») и как они обрабатываются.

Кто разработчик

Приложение разработано индивидуальным разработчиком (далее — «Разработчик»). Связь: terekh-spb@mail.ru.

Какие данные собирает само Приложение

Никаких персональных данных Приложение не собирает и не передаёт.

Внутри устройства Приложение хранит только локальный игровой прогресс (localStorage WebView): пройденные уровни, настройки звука/вибрации, статистика. Эти данные:

  • хранятся только на устройстве пользователя;
  • не передаются ни Разработчику, ни третьим лицам;
  • удаляются при удалении приложения (Android system reset);
  • могут быть очищены пользователем через системные настройки приложения.

Какие данные собирают встроенные сервисы

Приложение использует Yandex Mobile Ads SDK (рекламная сеть Яндекса) для показа межстраничной и вознаграждаемой рекламы. SDK работает по своей политике конфиденциальности и может собирать следующие данные с устройства пользователя:

  • Идентификатор рекламы Android (Android Advertising ID) — для отслеживания эффективности рекламных кампаний и предотвращения повторных показов.
  • IP-адрес — для геотаргетинга рекламы и определения региона.
  • Информация об устройстве — модель, версия Android, язык системы, разрешение экрана.
  • Информация о взаимодействии с рекламой — показы (impressions), клики, длительность просмотра.

Эти данные передаются на серверы ООО «Яндекс» и обрабатываются согласно политике Яндекса:

  • Политика конфиденциальности Яндекса
  • Политика обработки данных в Yandex Mobile Ads

Возрастные ограничения

Приложение предназначено для лиц старше 16 лет (категория 16+). Yandex Mobile Ads SDK может ограничивать тип показываемой рекламы для несовершеннолетних в соответствии с законодательством РФ.

Права пользователя

Пользователь имеет право:

  • Отключить персонализированную рекламу: настройки Android → Реклама → «Сброс рекламного идентификатора» / «Не персонализировать рекламу».
  • Удалить локальные данные: настройки Android → Приложения → «Правда или Действия: Игра для парочек 18+» → Хранилище → «Очистить данные», либо просто удалить приложение.
  • Связаться с Разработчиком по любым вопросам: terekh-spb@mail.ru.

Cookies, аналитика, серверный обмен

Приложение не использует:

  • Cookies (WebView хранит только localStorage внутри приложения).
  • Сервисы аналитики (Google Analytics, AppMetrica, Firebase и т.п.).
  • Собственный backend / API.
  • Авторизацию через email, телефон, соцсети.
  • Push-уведомления.
  • Геолокацию (кроме той, что определяется по IP силами Yandex Mobile Ads).

Изменения политики

Разработчик может обновить эту политику. Существенные изменения будут сопровождаться повышением версии приложения в магазине и публикацией новой версии этого файла в облачном хранилище (URL остаётся прежним).
