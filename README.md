# Правда или Действие — для пар

Лёгкая романтическая игра «Правда или Действие» для двоих взрослых людей в отношениях. Простой HTML/JS/CSS-проект без фреймворков, работает локально и легко оборачивается в APK.

## Что внутри

- 200+ карточек на русском языке (правды и действия).
- 5 режимов: Нежный, Флирт, Романтика, Смешной, Смешанный.
- Регулируемая интенсивность 1–5.
- 12 ходов за партию, бутылочка крутится и выбирает карточку.
- Каждую карточку можно один раз заменить.
- Заготовка под рекламу (interstitial и rewarded) с mock-режимом.
- Конфетти раз в 20 карточек, переворот и плавные анимации.
- Скрытая dev-панель `?dev=1`.

## Как запустить локально

Просто открыть `index.html` в браузере. Сервер не нужен — всё хранится в `localStorage`.

Опционально для удобства:

```bash
# Через любой простой HTTP-сервер, например:
python -m http.server 8080
# или
npx serve .
```

## Как добавлять карточки

Открой `cards.js` и добавь новую запись в массив `window.CARDS`:

```js
{
  id: 301,                    // уникальный, можно просто следующий по порядку
  type: 'truth',              // 'truth' | 'dare'
  category: 'нежные',         // см. список ниже
  intensity: 2,               // 1..5
  text: 'Текст карточки на русском.'
}
```

Категории:

- `нежные`
- `флирт`
- `романтические`
- `смешные`
- `для разговора`
- `лёгкие челленджи`

Запрещённое содержимое:
- 18+, насилие, унижение;
- алкоголь, деньги, опасные физические задания;
- темы измен, травм, болезней, политики, религии.

В dev-панели есть кнопка «Проверить контент» — она найдёт дубликаты id, пустые тексты и подозрительные слова из `config.js → blacklist`.

## Настройки рекламы

Все параметры в `config.js`:

| Параметр | Что делает |
| --- | --- |
| `cardsBeforeFirstAd` | Сколько карточек без рекламы в начале |
| `minCardsBetweenAds` / `maxCardsBetweenAds` | Случайный интервал между interstitial |
| `mockAds` | Включить mock-рекламу (для разработки) |
| `defaultMaxIntensity` | Стартовая интенсивность |
| `rewardedUnlockDuration` | На сколько ходов открывается категория после rewarded |
| `enableVibration` / `enableSound` | По умолчанию |

## Подключение Яндекс-рекламы

1. Подключите SDK Яндекс.Рекламы в `index.html`:
   ```html
   <script src="https://yandex.ru/ads/system/context.js" async></script>
   ```
2. В `ads.js` внутри функций `showInterstitialAd` и `showRewardedAd` замените блок mock на вызовы `Ya.Context.AdvManager.render(...)` с вашими block-id.
3. В `config.js` поставьте `mockAds: false`.

## Как обернуть в APK

Проект — статическая папка, её можно упаковать любым способом:

- **WebView-обёртка**: Cordova / Capacitor / любой Android-проект с `WebView`, у которого `loadUrl('file:///android_asset/index.html')`. Достаточно перенести содержимое папки в `assets/`.
- **Готовый билдер**: на Windows есть `Html2Apk Builder` — кладёшь папку и собираешь.
- **TWA**: если планируешь хост на https, можно сделать Trusted Web Activity.

Перед сборкой полезно:

- Проверить, что `mockAds = false` (если реклама уже подключена).
- Убедиться, что все пути в `index.html` относительные (так и есть).
- Перепроверить manifest/иконки на стороне обёртки.

## Безопасность контента

Игра рассчитана на пары. Любую карточку можно один раз заменить кнопкой «Заменить». Контент специально без 18+, без опасных и унижающих заданий.
